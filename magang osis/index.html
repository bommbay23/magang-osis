Games


Pada acara akan disediakan beberapa games yang seru dan menarik sehingga dapat menciptakan acara yang dinamis.

Criminal case


Criminal Case adalah permainan detektif di mana pemain menyelidiki TKP, menemukan petunjuk, memecahkan teka-teki, dan menangkap pelaku kejahatan dalam serangkaian kasus kriminal.

Api unggun


Kegiatan api unggun adalah acara berkumpul di sekitar api untuk bernyanyi, bercerita, bermain, dan mempererat kebersamaan.

Movie Night


acara menonton film bersama teman untuk bersantai, bersenang-senang, dan menikmati waktu kebersamaan.

Tree letter


"Tree Letter" adalah permainan kata di mana pemain harus menyusun kata-kata dengan tiga huruf yang berbeda untuk melatih kreativitas dan kecepatan berpikir.
